# Premier League ETL Pipeline

An ETL pipeline built as part of the Sparta Education Modern Data Engineering programme.

## What it does
- Extracts Premier League match data from football-data.org
- Transforms and cleans it into fact/dimension tables
- Loads into MongoDB
- Uploads a weekly summary to AWS S3
